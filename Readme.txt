	************************************
*****  	Description de l'application:   	*****
	************************************

ProxiBanqueSI_JL_JFA est une application de gestion de comptes bancaires
cette version vise a afficher et gerer le portefeuille Client d'un conseiller de la banque
- l'utilisateur interagit avec ProxyBanque a travers une interface web


	************************************
*****  	Livrables:			  				*****
	************************************
	
Vous trouverez les elements suivants:

1) la documentation au format javadoc : cf Documentation.

2) les sources du projet : dans le dossier 'sources\' de votre repertoire d'installation

3) les scripts SQL de creation des tables de la base de donnees : 
   ils ont au format MySQL, pour une Base MySQl (cf installation)

4) le fichier Readme.txt : ce document


	************************************
*****  	Documentation:			  			*****
	************************************

Veuillez suivre les indications suivantes:

Depuis votre navigateur, ouvrez le fichier 'index.html' 
situe dans le dossier 'javadoc\' de vote repertoire d'installation.

	
	************************************
***** 	Installation    			 		*****
	************************************

- Depuis l'interface de gestion de votre base de donnee MySQL :

1/ Executer le script 'ProxyBanque_V02_creationDB_MySQL.sql' 
   situe dans le repertoire 'SQL' de votre repertoire d'installation

2/ Charger les donnees test de la base : 
   Executer le script 'ProxyBanque_V02_BaseData_MySQL.sql' 
   situe dans le repertoire 'SQL' de votre repertoire d'installation
